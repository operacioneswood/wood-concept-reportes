Instrucciones de exportación
ClickUp CSV
  1. Abre la vista de lista con todas las tareas
  2. Clic en "···" → Export → CSV
  3. Incluye: Task Type, Parent Name, NO. OP, Nivel, Assignee, FIN DE DIBUJO DE APROBACIÓN, ENVÍO A FÁBRICA
  4. Sube el archivo aquí
Registro de Entrada (Excel)
  1. Abre el Registro de Entrada en Excel
  2. Archivo → Guardar como → CSV (delimitado por comas)
  3. Columnas necesarias: OP, NIVEL, ENTRADA, DISEÑADOR, DESCRIPCIÓN, CLIENTE
  4. Sube el archivo aquí
